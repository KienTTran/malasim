Detailed Description

Encapsulates random number generation functionalities using GSL.

The Random class provides various methods to generate random numbers following different distributions. It manages a GSL random number generator (RNG) instance, ensuring proper resource management and offering a clean interface for simulations.

inline

Generates a truncated normally distributed random number within a specified number of standard deviations.

The function repeatedly generates normally distributed random numbers until one falls within the range [mean - truncation_limit * standard_deviation, mean + truncation_limit * standard_deviation]. To prevent potential infinite loops, a maximum number of attempts is enforced.

Template Parameters
TThe return type, typically double or int.
Parameters
meanThe mean of the normal distribution.
standard_deviationThe standard deviation of the normal distribution.
truncation_limitThe number of standard deviations to truncate (default is 3.0).
max_attemptsThe maximum number of attempts to generate a valid truncated value (default is 1000).
Returns
T A truncated normally distributed random number.
Exceptions
std::runtime_errorIf RNG is not initialized or if a valid truncated value cannot be generated within the maximum attempts.

template<typename T >
T Random::random_uniform ( T from,
T to )
inline

Generates a uniformly distributed random number within a specified range.

This function returns a random number of type T that is uniformly distributed between the specified from and to values. The range is inclusive of from and exclusive of to. It supports both integral and floating-point types.

Template Parameters
TThe numeric type of the random number to generate. Must be an arithmetic type.
Parameters
fromThe lower bound of the range. Must be less than to.
toThe upper bound of the range. Must be greater than from.
Returns
A random number of type T uniformly distributed in [from, to).
Exceptions
std::runtime_errorIf the random number generator is not initialized.
std::invalid_argumentIf from is not less than to.
Note
Ensure that the random number generator (rng_) is properly initialized before invoking this function.

template<typename T >
void Random::shuffle ( std::vector< T > & vec) const
inline

Shuffles the elements of a vector in place using the current random number generator.

This templated function randomizes the order of elements within the provided std::vector<T> using GSL's gsl_ran_shuffle function. It ensures that the shuffling process maintains the integrity of the data, meaning no elements are lost or duplicated during the operation.

Template Parameters
TThe type of elements stored in the vector. Can be any copyable and movable type.
Parameters
vecA reference to the std::vector<T> whose elements are to be shuffled. Must not be empty.
Exceptions
std::runtime_errorIf the random number generator (rng_) is not initialized.
std::invalid_argumentIf the provided vector vec is empty.
Note
  • The function requires that the random number generator (rng_) is properly initialized.
  • The randomness of the shuffle is dependent on the underlying RNG's quality and seeding.
Warning
  • Ensure that the Random class instance has been initialized with a valid RNG before calling this function.
